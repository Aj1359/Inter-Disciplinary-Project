================================================================================
QUICK SUMMARY: WHY BRUTE FORCE BC REDUCTION VARIES ACROSS GRAPHS
================================================================================

SIMPLE ANSWER:
The brute force algorithm's reduction percentage varies because:
1. Different graphs have different structures
2. Different structures have different edge criticality patterns
3. Removing critical edges = higher reduction
4. Removing non-critical edges = lower reduction

DETAILED EXPLANATION BY GRAPH TYPE
================================================================================

GROUP 1: HIGHLY STABLE (0% Variance)

PATH GRAPH & BARBELL: 100% Reduction
  Why: Linear structure or bridge connects two complete graphs
  Result: ALWAYS the same best edge (the bridge or center edge)
  Variance: 0% (identical every trial)

STAR GRAPH: 0.58% Reduction
  Why: Center hub connects to N peripheral nodes
  Result: Peripheral edges have no paths between periphery (low BC)
  Variance: 0% (identical every trial)

CAVEMAN: 5.57% Reduction
  Why: Multiple complete cliques with bridge edges
  Result: Bridge edges are critical; within-clique edges redundant
  Variance: ~0% (highly predictable structure)

GROUP 2: MODERATELY VARIABLE (0.5-1.5% Variance)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

BARABASI-ALBERT: 4.17% Reduction ± 0.61%
  Why: Random seed creates different graph each time
  Range: 3.15% - 5.29%
  
  Mechanism:
    • Sometimes target node is near hub (high reduction)
    • Sometimes target node is in periphery (low reduction)
    • Hub edges control 10-100x more paths than peripheral edges
    
  Variance Source: Random graph generation with different seeds

POWERLAW-CLUSTER: 3.96% Reduction ± 0.98%
  Why: Mix of power-law distribution + clustering
  Range: 2.74% - 6.06%
  
  Mechanism:
    • Inter-cluster edges: high BC (break many shortest paths)
    • Within-cluster edges: low BC (alternative paths exist)
    • Random rewiring creates different graph structures
    
  Variance Source: Random components + cluster connectivity

ERDOS-RENYI: 5.92% Reduction ± 1.20%
  Why: Completely random graph, no structure
  Range: 3.83% - 7.80%
  
  Mechanism:
    • All edges have relatively similar importance
    • No hubs or bridges
    • Paths distributed across many edges
    • Removing any single edge has modest impact
    
  Variance Source: Random connectivity differences between trials

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

GROUP 3: HIGHLY UNSTABLE (2-15% Variance)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TREE: 50.80% Reduction ± 13.93%
  Why: TREE STRUCTURE VARIES RANDOMLY EACH TRIAL
  Range: 32.99% - 87.44%
  
  Mechanism:
    • Each seed creates different tree shape
    • Deep trees: long paths → removing edge breaks many paths → high reduction
    • Shallow trees: short paths → removing edge breaks fewer paths → low reduction
    • Different node degrees affect BC distribution
    
  Variance Source: Extremely high - tree topology completely changes per seed

WATTS-STROGATZ: 6.20% Reduction ± 3.44%
  Why: REWIRING CREATES COMPLETELY DIFFERENT GRAPHS
  Range: 1.22% - 14.27%
  
  Mechanism:
    • Start with ring (deterministic)
    • Randomly rewire edges (creates shortcuts)
    • Sometimes shortcuts create critical bridges (high reduction)
    • Sometimes shortcuts create redundancy (low reduction)
    • Each trial rewiring is different!
    
  Extreme Cases:
    • Trial A: Rewiring creates 2-3 critical shortcut edges (14% reduction)
    • Trial B: Rewiring creates local redundancy (1% reduction)
    • Trial C: Ring structure remains mostly intact (3% reduction)
    
  Variance Source: HIGHEST - rewiring algorithm completely recreates graph

================================================================================
THE 5 KEY FACTORS DRIVING VARIATION
================================================================================

1. GRAPH DENSITY (edges per node)
   ─────────────────────────────
   Sparse graphs (trees, paths):
   → Fewer alternative paths
   → Removing edge has bigger impact
   → Higher reduction, less variable
   
   Dense graphs (random, complete):
   → Many alternative paths
   → Removing edge is less critical
   → Lower reduction, more variable

2. EDGE CRITICALITY
   ────────────────
   Star/Hub graphs:
   → Center hub edge: critical for almost ALL paths
   → Peripheral edges: critical for NONE
   → High variance in edge importance
   
   Random graphs:
   → All edges have similar importance
   → No critical edges
   → Low variance in edge importance

3. CENTRALITY DISTRIBUTION
   ──────────────────────
   Concentrated BC:
   → Few edges carry most shortest paths
   → Removing them = high reduction
   → Removing others = low reduction
   
   Distributed BC:
   → All edges carry paths fairly evenly
   → No single edge has huge impact

4. GRAPH GENERATION METHOD
   ───────────────────────
   Deterministic (fixed algorithm):
   → Path: 0-1-2-3-...-n (always same)
   → Result: 0% variance
   
   Random seed (different graph each time):
   → Barabasi with seed=1 ≠ Barabasi with seed=2
   → Result: HIGH variance
   
   Rewiring (recreates structure):
   → Watts-Strogatz rewires each trial
   → Result: EXTREME variance

5. DEGREE DISTRIBUTION
   ───────────────────
   Uniform degrees:
   → All nodes similar importance
   → Balanced connectivity
   → Lower reduction variance
   
   Power-law (few hubs + many periphery):
   → Hub edges: 10-100x more important
   → Peripheral edges: mostly insignificant
   → High reduction variance

================================================================================
MATHEMATICAL FORMULA
================================================================================

BC(edge) = (# shortest paths using edge) / (total shortest paths)
Reduction = BC(target_node) - BC(after_removal) / BC(target_node)

Variance(reduction) ∝ Variance(BC_distribution) × Variance(graph_structure)

Examples:
─────────
Path Graph:
  Variance(BC_dist) = LOW     (all edges equal)
  Variance(structure) = 0     (always linear)
  → Variance(reduction) = 0%
  
Barabasi-Albert:
  Variance(BC_dist) = HIGH    (hubs >> periphery)
  Variance(structure) = HIGH  (different seeds)
  → Variance(reduction) = MODERATE (0.61%)
  
Watts-Strogatz:
  Variance(BC_dist) = EXTREME (shortcuts random)
  Variance(structure) = EXTREME (rewiring random)
  → Variance(reduction) = EXTREME (3.44%)

================================================================================
PRACTICAL IMPLICATIONS
================================================================================

When to Expect CONSISTENT results:
  ✓ Path, Star, Barbell, Caveman graphs
  ✓ Deterministic/fixed structure graphs
  ✓ Same edge will be selected every trial
  
When to Expect VARIABLE results:
  ✗ Barabasi-Albert, Erdos-Renyi graphs
  ✗ Random seed generates different graphs
  ✗ Different optimal edges might be selected
  
When to Expect HIGHLY VARIABLE results:
  ✗✗ Watts-Strogatz (rewiring creates new graphs)
  ✗✗ Tree (different random seeds create different shapes)
  ✗✗ May find completely different edges
